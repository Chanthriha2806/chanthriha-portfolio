# Quick Start - One Page

## 🚀 Installation (30 seconds)

### Windows
1. Run `INSTALL.bat`
2. Run `RUN.bat`
3. Open: **http://localhost:8080**

### Mac/Linux
```bash
bash INSTALL.sh
bash RUN.sh
```

## ⚠️ Common Issues

**Java not found?**
→ Download from java.com

**Port 8080 in use?**
→ Kill process or edit HelpDeskServer.java

**Login fails?**
→ Clear browser cookies, use exact credentials

**Data not saving?**
→ Login as admin first, check folder permissions
